题解 P4981父子 (Carley定理)

Posted on 2021-02-20，4 min read

原题链接

题意

构造一棵含n个带序号的点的有根树，求方案数

Carley定理~
Carley定理：n个带序号的节点共能构成nn−2n^{n-2}nn−2个形态不同的无根树

套！

Carley定理处理的是无根树，而这题是有根树；树的每种形态都有n个点可做根，所以总方案数为nn−1n^{n-1}nn−1

核心代码

int main()
{
	int t;
	read(t);
	F(i, 1, t)
	{
		ll n;
		read(n);
		write(qpow(n, n-1), '\n');
	}
	return 0;
}

using namespace fastIO;
ll mod = 1e9 + 9;

ll qpow(ll n, ll k)
{
	ll tmp = 1;
	while(k)
	{
		if(k & 1)
		{
			tmp *= n;
			tmp %= mod;
		}
		k >>= 1;
		n *= n;
		n %= mod;
	}
	return tmp;
}
